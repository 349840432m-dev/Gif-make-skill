# CineGIF 变更记录

本文件记录面向产品、接口、配置、流程和兼容性的实际变化。版本日期使用 `YYYY-MM-DD`，未完成事项不得写入“已完成”。

## [Unreleased]

### 文档

- 2026-06-15：建立 CineGIF 最小完整文档链。
- 明确产品为中文、本地优先的单图动效、多图轮播和视频转 GIF 生成器。
- 明确技术基线为 React、TypeScript、Vite、Canvas 和浏览器 GIF 导出。
- 固定三步流程：导入素材、编辑效果、导出 GIF。
- 固定六种动效：电影推进、轻盈漂浮、呼吸脉动、左右摇曳、环绕镜头、景深视差。
- 明确中文深色三步引导式视觉与核心失败路径。

### 当前范围

- 单张 JPEG、PNG、WebP 导入。
- 2-10 张图片多图轮播、追加、排序、删除。
- 1 个本地 MP4、WebM 或 MOV 视频导入并截取片段转 GIF。
- 六种预设动效、强度与时长。
- 直接切换、淡入淡出、横向滑动三种多图转场。
- 视频开始/结束时间、帧率、输出宽度、画面比例和循环设置。
- Canvas 循环预览。
- 浏览器内 GIF 编码、取消、进度、下载与重试。
- 图片和输出默认不上传、不持久化。

### 当前不做

- 音频、文字、滤镜、图层、时间轴和多视频拼接。
- 账号、云存储、跨设备同步、作品库和分享。
- 服务端编码及 MP4、WebM、APNG 导出。
- 移动端专项体验、浅色主题和插件系统。

### 已知失败路径

- 非法、损坏、超大或无法解码的图片必须在入口拦截。
- Canvas 或编码能力不足时必须明确阻止并显示可恢复错误，不得静默失败。
- 预览、编码、超时、内存不足、取消和下载失败后必须释放资源并保留可恢复参数。

### 新增

- 2026-06-18：补充 Node.js `^20.19.0 || >=22.12.0`、npm 包管理器和可移植忽略规则，使项目可作为自包含 Skill 模板通过 `npm ci` 在新电脑复现。
- 完成图片上传、Canvas 实时预览、播放暂停、参数调整、进度取消和 GIF 下载流程。
- 新增“单图动效 / 多图轮播 / 视频转 GIF”三模式；多图支持 2-10 张图片、统一画布、三种转场、排序、删除、追加和本地 GIF 导出。
- 增加多图素材总大小、总像素、GIF 总帧数预算与对应测试。
- 新增“视频转 GIF”第三模式；视频支持本地 MP4、WebM、MOV、100 MB 上限、8 秒片段、240 帧预算、原生 `video` + Canvas 抽帧和本地 GIF 导出路径。
- GIF 编码器支持异步逐帧绘制，以便视频导出在每帧 seek 后写入。
- 重排 UI：模式选择移动到顶部右侧，三步流程移动到左侧窄栏并线性展示，中间功能区按当前流程切换上传、设计和导出内容，实时画布移动到最右侧。
- 视频预览改为播放原生 `video` 并持续绘制到 Canvas，减少“看起来停在首帧”的体验；视频 GIF 导出仍按帧 seek 抽帧。
- 将单图、多图和视频的全局预览/导出参数从左侧流程功能区移到实时画布下方，常态展示并按宽画布区域多列排版。
- 默认输出宽度提升到 1280 px，并为当前素材提供“原素材宽度”选项，降低 GIF 导出尺寸被默认缩小的问题。
- 顶栏新增“清空重置”按钮，用于释放所有素材和导出结果并恢复初始状态。
- 视频参数新增轻量双端点时间轴，可拖动开始/结束节点并跳到对应视频画面，用于按时间节点选择 GIF 片段。
- 视频参数新增播放速度控件，支持慢放和快放；预览使用 `video.playbackRate`，导出按速度改变源视频采样步长。
- 使用 `gifenc@1.0.3` 本地编码，选型记录见 `ADR/001-gif-encoder.md`。
- 增加 80 项自动化测试，并通过类型检查、lint 和生产构建。

### 未验证

- Safari、Firefox、Edge、JPEG/WebP 实文件及低内存设备仍待扩大手工验收。

### 修复

- 修复 1280 px 桌面宽度下导出按钮横向溢出；桌面 `scrollWidth` 已回落至视口以内。
- 修复隐藏脚本触发下载在部分浏览器中无响应的问题；导出完成后显示原生“下载 GIF”链接，并增加“新标签页打开”兜底入口。

### 验证

- Chromium 中完成上传、实时播放/暂停、六动效选择、参数切换、GIF89a 编码、取消、下载文件名、错误 MIME、桌面和 390 px 响应式验收。
- Chromium 中完成多图模式切换、导入、排序、删除、追加、横向滑动转场、GIF89a 编码、下载文件名和单图回归验收。
- Chrome 149 中完成视频真实上传、Canvas 预览、GIF89a 导出和下载验收；修复视频预览极小负浮点时间导致的 seek 错误。
